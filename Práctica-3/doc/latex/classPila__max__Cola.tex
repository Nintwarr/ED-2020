\doxysubsubsection{Descripción detallada}
T.\+D.\+A. Pila\+\_\+max. 

Una instancia {\itshape c} del tipo de dato abstracto {\ttfamily \mbox{\hyperlink{classPila__max__Cola}{Pila\+\_\+max\+\_\+\+Cola}}} es un objeto que representa una estructura de datos que funciona tal cual funcionan las colas tradicionales. La peculiaridad de esta cola es que guarda parejas de valores. Así, el primer elemento es el elemento normal con el que trabajamos y el segundo representa el valor máximo introducido en la cola hasta el momento.
