Descripción detallada

T.D.A. Pila_max.

Una instancia c del tipo de dato abstracto Pila_max_Cola es un objeto que representa una estructura de datos que funciona tal cual funcionan las colas tradicionales. La peculiaridad de esta cola es que guarda parejas de valores. Así, el primer elemento es el elemento normal con el que trabajamos y el segundo representa el valor máximo introducido en la cola hasta el momento.

Para esta versión en concreto, se utiliza un T.D.A. cola propio

@section invPila_max_cola Invariante de la representación

El invariante es:

Para todo rep.cola.frente().ele = e, entonces rep.cola.frente().maximo >= e. 

@section faPila_max_Cola Función de abstracción

Un objeto válido @e rep del TDA Cola_max_Cola representa a una

estructura de datos que se comporta como una pila que almacena datos del tipo elemento. Estos son parejas valor-maximo.

Si la pila tiene n elementos y el tope es el elemento n-ésimo: | rep.cola[n] | Corresponde con rep.cola.ultima. | rep.cola[n-1] | | ... | | rep.cola[0] | Corresponde con rep.cola.primera.
